This page presents an interactive bubble chart showing experimental validation of autism-associated genes across different model systems. Each bubble represents a unique gene‚Äìmodel pair. The size of each bubble indicates the number of published functional studies (based on PubMed IDs), and the color indicates the type of model organism used. Data are compiled from manual curation and sources including Fu et al. (2022, Nature Genetics) [PMID: 35982160]. Click on any bubble to view associated PMIDs with direct links to PubMed.

  • X-axis: Gene symbol
  • Y-axis: Model organism (e.g., Mouse, Zebrafish, Organoid)
  • Bubble size: Number of functional studies (PMIDs)
  • Bubble color: Model type
  • Search box: Filter by gene or model name
